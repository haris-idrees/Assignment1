Follow the given steps to upload this file using terminal

Step 1:

Create a repository on github like Assignment1

Step 2:

Create a folder and add all the files that you want to upload to the repository


In my case I created a folder named Assignment1 and added a file index.html

Step 3:

Initialize the git by typing Git init

Step 4:

After initializing set your global identity using the given commands:


git config --global user.email "xyz@gmail.com"
git config --global user.name "your name"

Step 5:

Check the status of the gig using command git status


Step 6:

Type Git add filename.extension or Git add.(to add all files) to add the files in repository


Step 7:

Type Git commit -m"Message" to save current changes on local system


Step 7:

Type Git push to push all the data on online repository
